## Грабли по работе с Drive

1. `search_files` **не поддерживает** `'X' in parents` — только
   `parentId = 'X'`.
2. `read_file_content` для Google Sheets **смешивает листы** в один
   markdown. Для конкретного листа — скачивать xlsx через
   `download_file_content` и парсить через openpyxl.
3. **Операции delete через MCP нет.** Лишние файлы пользователь
   удаляет руками через интерфейс Drive.
4. `mkdir -p {a,b,c}` через bash не разворачивает фигурные скобки —
   создавать папки отдельными вызовами.
