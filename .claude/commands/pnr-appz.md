---
description: Рассчитать ПНР АППЗ для объекта (22 наряда по методике Рубеж)
---

Запусти сабагента `pnr-appz` для расчёта ПНР АППЗ.

Аргументы пользователя (имя объекта / спецификация / география): $ARGUMENTS

Если каких-то данных не хватает (спецификация, география, имя объекта, наличие Firesec, разбивка ШУВ по ЧП, количество вентиляторов/калориферов на ШУВ) — сабагент должен спросить, не угадывать.
