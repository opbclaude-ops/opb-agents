Тонкая обёртка над навыком `appz-kvartirograph` v1.0. Источник истины — `SKILL.md` навыка. Здесь — только транспорт.

## Шаги

1. **Понять вход.** Что прислал пользователь:
   - ссылка/id папки Drive,
   - локальный путь (`/mnt/user-data/uploads/...` или `/home/claude/...`),
   - приложенный PDF.
   Если непонятно — один уточняющий вопрос.

2. **Скачать материалы в `/home/claude/<имя_проекта>/`.**
   - Для Drive: `search_files` по `parentId = '<id>'` → `download_file_content` по нужным PDF.
   - Для локального — проверить, что файлы на месте.

3. **Убедиться, что навык `appz-kvartirograph` доступен.** Порядок поиска:
   1. Локально: `~/.claude/skills/appz-kvartirograph/` или `/mnt/skills/user/appz-kvartirograph/`.
   2. Fallback с Drive: папка `Skills/appz-kvartirograph/v1.0/` (родитель `Skills` — id `1Qh8-R0okEZTPF_zHVRUPGCRhwSh7jqQu`). Выгрузить через `download_file_content`, восстановить структуру в `/home/claude/.skills/appz-kvartirograph/` (SKILL.md + references + assets; scripts — копии из `appz-shahmatka`, взять оттуда).
   - Если по этому же объекту есть `appz-shahmatka` (локально или на Drive) — навык опционально использует её как ускорение. Без жёсткой зависимости: нет шахматки — собирает мини-контекст сам.

4. **Прочитать `SKILL.md` навыка целиком.** Дальше — строго по нему: 6 проходов (записка → структурка → судья → план типового → нетиповые → сверка с интернет-источниками), ворота на каждом этапе, креативный поиск как сквозной принцип.

5. **Запустить навык.** Не вмешиваться в логику, не сокращать этапы, не подменять самопроверки. Если навык требует уточнения — пробрасывать вопрос пользователю как есть.

6. **Получить результат** — таблица квартирографии посекционно и поэтажно + раздел «Гипотезы новых методов» (если найдены).

7. **Отдать через `present_files`.** Краткая сводка: объект, секций × этажей, итог по типам квартир, флаги «требует уточнения», сошлась ли сверка с интернет-источниками.

## Принципы

- Навык — источник истины, агент — транспорт. Хочется поправить логику — правится `appz-kvartirograph`, не этот файл.
- Триггеры в `description` агента = триггеры из `description` навыка.
- Самопроверки навыка не дублировать и не обходить.
- Версия привязана: `agent-kvartirograph v1.0` ↔ `appz-kvartirograph v1.0`. Появится `v1.1` навыка — обновим обёртку.
